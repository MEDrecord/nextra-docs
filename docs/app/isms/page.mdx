# Information Security Management System

Welcome to the MEDrecord ISMS documentation. This system documents our approach to managing information security in compliance with ISO 27001.

## Scope

The ISMS applies to:

- MedSafe PGO
- HealthTalk
- ClubDiabetes
- Supporting infrastructure and personnel
