Currently, we are considering the following changes. All changes follow our change management procedure.

## Change Template

When registering a new change, use the following template:

### Change Details

| Field | Description |
|-------|-------------|
| Initiator | Who initiated the change |
| Description | What the change involves |
| Rationale | Why this change is needed |

### Impact Assessment

| Area | Impact Level |
|------|--------------|
| Personnel | L/M/H |
| Customers | L/M/H |
| Assets | Description of affected assets |

### Risk Assessment

| Threat | Likelihood | Impact | Exposure | Treatment |
|--------|------------|--------|----------|-----------|
| Risk description | L/M/H | L/M/H | L/M/H | Mitigation approach |

### Rollback Plan

Document the rollback procedure in case the change needs to be reverted.
