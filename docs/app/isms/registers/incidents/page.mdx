# Incidents

All incidents are handled and reported as tickets inside our daily ticketing system.

## Current Incidents

Incidents that are currently being treated can be found in the ticketing system board.

## Archived Incidents

An archive of treated incidents can be found in the incident tracking system.

## Reporting an Incident

To report a security incident:

1. Create a ticket in the incident management system
2. Assign appropriate priority based on impact
3. Follow the [Incident Response Procedure](/helpdesk/isms/incident-response-procedure)
4. Document all actions taken
5. Complete post-incident review
