Welcome to the MEDrecord Security Awareness training. This section contains essential security knowledge for **all employees**.

## Why Security Matters

As a healthcare technology company, we handle sensitive patient data. Every employee plays a critical role in protecting this information and maintaining compliance with:

- **ISO 27001** - Information Security Management
- **NEN 7510** - Healthcare Information Security (Netherlands)
- **GDPR/AVG** - Data Protection Regulations
- **ISO 13485** - Medical Device Quality Management

## Quick Security Guidelines

| Do | Don't |
|----|-------|
| Lock your screen when away | Share passwords |
| Report suspicious emails | Click unknown links |
| Use strong, unique passwords | Use public WiFi for work |
| Encrypt sensitive files | Store data on personal devices |
| Verify caller identity | Discuss patient data in public |

## Report Security Issues

**See something suspicious?** Report it immediately:

1. **Phishing emails**: Forward to `security@medrecord.nl`
2. **Security incidents**: Use the [Incident Report Form](/isms/report-incident)
3. **Urgent issues**: Contact the Security Officer directly

## Annual Training Requirements

All employees must complete:
- Initial security awareness training (within first week)
- Annual refresher training
- Role-specific training (as applicable)

Training completion is tracked in MS Teams. Contact HR if you need access to training materials.
