## Control Details

| Field | Value |
|-------|-------|
| **Requirement** | Information security events shall be reported through appropriate management channels as quickly as possible. |
| **Applicable** | Yes |
| **Status** | Implemented |

## Implementation

- Clear reporting channels established for all personnel
- 24/7 incident reporting capability
- Automated alerting for critical security events
- Integration with ticketing system for tracking

## Reporting Channels

1. **Internal**: Create ticket in incident management system
2. **External (customers)**: Contact support or security email
3. **Emergency**: Direct escalation to Information Security Officer

## Check Details

| Field | Value |
|-------|-------|
| **Owner** | Information Security Officer |
| **Frequency** | Monthly |
| **Instruction** | Review incident reports and ensure timely escalation |
