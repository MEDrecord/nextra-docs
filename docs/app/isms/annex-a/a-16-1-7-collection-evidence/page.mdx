# A.16.1.7 Collection of evidence

## Control Details

| Field | Value |
|-------|-------|
| **Requirement** | The organization shall define and apply procedures for the identification, collection, acquisition and preservation of information, which can serve as evidence. |
| **Applicable** | Yes |
| **Status** | Implemented |

## Implementation

- Evidence collection procedures documented
- Chain of custody maintained for all evidence
- Logs retained according to retention policy
- Forensic tools available for investigation

## Evidence Types

| Type | Retention | Storage |
|------|-----------|---------|
| System logs | 12 months | Centralized logging system |
| Access logs | 36 months | Encrypted storage |
| Security events | 36 months | SIEM |
| Incident documentation | 7 years | Document management system |

## Check Details

| Field | Value |
|-------|-------|
| **Owner** | Information Security Officer |
| **Frequency** | Quarterly |
| **Instruction** | Verify log retention and evidence collection procedures |
