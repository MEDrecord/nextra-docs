## Control Details

| Field | Value |
|-------|-------|
| **Requirement** | Information about technical vulnerabilities of information systems being used shall be obtained in a timely fashion, the organization's exposure to such vulnerabilities evaluated and appropriate measures taken to address the associated risk. |
| **Applicable** | Yes |
| **Status** | Implemented |

## Implementation

- Automated dependency scanning in CI/CD pipelines
- Regular security updates applied to all systems
- Vulnerability disclosure process in place
- Third-party penetration testing conducted annually

## Check Details

| Field | Value |
|-------|-------|
| **Owner** | Lead Developer |
| **Frequency** | Continuous (automated) + Monthly review |
| **Instruction** | Review vulnerability scan results and ensure all critical/high vulnerabilities are addressed within SLA |
