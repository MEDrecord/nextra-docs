# Troubleshooting

Solutions to common issues and problems.

## Login Issues

### I cannot log in

**Possible causes:**
- Incorrect email or password
- Account locked due to too many failed attempts
- Browser cache issues

**Solutions:**
1. Double-check your email address for typos
2. Use the "Forgot password" link to reset your password
3. Clear your browser cache and cookies
4. Try a different browser or incognito mode
5. Contact support if your account may be locked

### I am not receiving the password reset email

**Solutions:**
1. Check your spam/junk folder
2. Verify you are using the correct email address
3. Wait a few minutes and try again
4. Add our domain to your email allowlist
5. Contact support for manual password reset

### Two-factor authentication is not working

**Solutions:**
1. Ensure your device time is synchronized
2. Try the backup codes provided during setup
3. If using an authenticator app, remove and re-add the account
4. Contact support for 2FA reset

## Messaging Issues

### Messages are not being delivered

**Check the following:**
1. Verify the recipient contact information is correct
2. Check if the recipient has opted out of communications
3. Ensure your messaging quota has not been exceeded
4. Review the message for content that may trigger spam filters

**If messages still fail:**
1. Check the delivery status in the message history
2. Look for error codes in the delivery report
3. Contact support with the message ID for investigation

### Message templates are not saving

**Solutions:**
1. Ensure all required fields are completed
2. Check for special characters that may not be supported
3. Verify your permissions allow template creation
4. Try saving with a different name

## Integration Issues

### Calendar sync is not working

**Solutions:**
1. Disconnect and reconnect the calendar integration
2. Check that you have granted all required permissions
3. Verify the calendar service is not experiencing outages
4. Review sync settings for correct configuration

### API calls are returning errors

See [Gateway Documentation](/docs/products/gateway) for detailed error code explanations and resolution steps.

## Performance Issues

### The application is running slowly

**Try these steps:**
1. Clear your browser cache
2. Disable browser extensions temporarily
3. Check your internet connection speed
4. Try a different browser
5. Check our status page for known issues

### Reports are taking too long to generate

**Solutions:**
1. Reduce the date range of your report
2. Apply more specific filters
3. Schedule large reports for off-peak hours
4. Contact support for assistance with large data exports

## Still Need Help?

If you cannot resolve your issue:

1. **Check status page** - Verify there are no ongoing incidents
2. **Search documentation** - Use the search bar to find related topics
3. **Contact support** - Use the Helpdesk chat for immediate assistance
4. **Report a bug** - Use the issue reporter in your application
