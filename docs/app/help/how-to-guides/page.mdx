# How-to Guides

Step-by-step instructions for common tasks.

## Getting Started

### Set up your first communication workflow

1. Log in to your dashboard
2. Navigate to **Workflows** in the sidebar
3. Click **Create New Workflow**
4. Select a template or start from scratch
5. Configure triggers, actions, and recipients
6. Test your workflow with a sample message
7. Activate when ready

### Connect your calendar for appointments

1. Go to **Settings** then **Integrations**
2. Find your calendar provider (Google, Outlook, etc.)
3. Click **Connect** and authorize access
4. Configure sync preferences
5. Your appointments will now appear in HealthTalk

### Send your first patient message

1. Navigate to **Messages** in the sidebar
2. Click **New Message**
3. Search for and select the recipient
4. Choose a message template or write a custom message
5. Select the delivery channel (SMS, email, etc.)
6. Review and click **Send**

## Advanced Tasks

### Create a multi-step appointment reminder

1. Go to **Workflows** then **Create New**
2. Add trigger: "Appointment scheduled"
3. Add action: Send reminder 24 hours before
4. Add action: Send reminder 2 hours before
5. Add action: Send follow-up after appointment
6. Configure message content for each step
7. Test and activate

### Export patient communication history

1. Navigate to **Reports** then **Communication History**
2. Set date range and filters
3. Select export format (CSV, PDF)
4. Click **Export**
5. Download the generated file

### Configure notification preferences

1. Go to **Settings** then **Notifications**
2. Toggle which events trigger notifications
3. Choose delivery methods for each notification type
4. Set quiet hours if needed
5. Save changes
