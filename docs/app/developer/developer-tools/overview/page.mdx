## Core Principles

### Better Prompts = Better Results

The quality of AI output depends on how clearly you communicate. Good prompts deliver:
- Cleaner, more maintainable code
- Smarter UX decisions
- 30-40% faster generation with fewer iterations

### Consistent Tooling

Same tools, same patterns across all projects:
- Standardized project rules across repositories
- Shared custom commands for common tasks
- Unified CI/CD workflows

### AI Augmentation, Not Replacement

Use AI to accelerate development while maintaining quality:
- AI handles boilerplate and routine code
- Humans review, validate, and make architectural decisions
- Tests and code review remain essential
