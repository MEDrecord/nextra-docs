## Philosophy

Modern development teams benefit from:

- **AI augmentation** - Use AI to accelerate development while maintaining quality
- **Consistent tooling** - Same tools, same patterns across all projects
- **Documentation as code** - Keep docs close to code, version-controlled
- **Automation first** - Automate repetitive tasks to focus on creative work
