# API Reference Overview

The MEDrecord Gateway API enables programmatic access to messaging, scheduling, and patient data across all platform products (HealthTalk, MedSafe, Coachi).

## Base URL

```
https://auth.healthtalk.ai/v1
```

> **Note:** While the base URL contains `healthtalk.ai`, this is the unified gateway for all MEDrecord products.

## API Features

- **RESTful Design** - Standard HTTP methods and status codes
- **JSON Format** - Request and response bodies in JSON
- **HMAC Authentication** - Secure signature-based authentication
- **Rate Limiting** - Fair usage policies to ensure service availability
- **Webhooks** - Real-time event notifications

Official SDKs are available for:

- Python
- C# / .NET

## Versioning

The API uses URL-based versioning. The current version is `v1`. Breaking changes will result in a new version number.
