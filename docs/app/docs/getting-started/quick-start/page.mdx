# Quick Start

Get up and running with HealthTalk in minutes.

## Step 1: Log In

Access HealthTalk at your organization's URL and log in with your credentials.

## Step 2: Send Your First Message

1. Navigate to **Messages** in the sidebar
2. Click **New Message**
3. Search for a patient by name or ID
4. Select a message template or compose a custom message
5. Review the message preview
6. Click **Send**

## Step 3: Schedule an Appointment Reminder

1. Navigate to **Scheduling**
2. Select an appointment from the calendar
3. Click **Send Reminder**
4. Choose the reminder timing (e.g., 24 hours before)
5. Confirm the delivery channel (SMS, email, or both)

## Step 4: View Reports

1. Navigate to **Analytics**
2. View message delivery statistics
3. Check response rates
4. Export reports for compliance documentation

## Common Tasks

### Create a Message Template

1. Go to **Settings > Templates**
2. Click **New Template**
3. Enter template name and content
4. Add dynamic fields (patient name, appointment date, etc.)
5. Submit for approval (if workflow enabled)

### Add a New User

1. Go to **Settings > Users**
2. Click **Add User**
3. Enter user details and assign role
4. User receives invitation email
