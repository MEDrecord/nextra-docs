# ConsortiaAI

ConsortiaAI is an AI-powered research grant and consortium management platform for coordinators of European and national R&D projects. It supports the full lifecycle from funding discovery and partner matching to proposal development, consortium agreements, milestone tracking, deliverables, compliance, and reporting.

## Overview

ConsortiaAI helps research coordinators manage complex multi-partner projects efficiently. The platform uses AI agents to automate repetitive tasks while keeping humans in control of strategic decisions.

### Target Users

- Research coordinators at University Medical Centers
- Research Support Office staff
- Project managers for Horizon Europe, Eurostars, ZonMw, NWO projects
- Innovation managers at healthcare organizations

### Key Capabilities

- **Funding Intelligence** - AI-powered call monitoring and eligibility matching
- **Consortium Builder** - Partner discovery, outreach automation, LOI tracking
- **Proposal Workspace** - Collaborative editing with compliance checks
- **Contracts and Agreements** - Templates for CA, DPA, IP schedules with e-signatures
- **Planning and Scheduling** - Cal.com integration for partner meetings and deadlines
- **Project Execution** - Work packages, milestones, deliverables tracking
- **Financials and Payments** - Budget tracking, success fees, invoicing via Stripe
- **Reporting Suite** - Periodic reports, financial reporting, KPI dashboards

## Planning and Scheduling (Cal.com)

ConsortiaAI integrates with Cal.com for consortium scheduling and coordination:

### Features

| Feature | Description |
|---------|-------------|
| **Partner Meetings** | Round-robin booking with multiple coordinators |
| **Kick-off Planning** | Team scheduling with availability overlap detection |
| **Review Deadlines** | Recurring event types for periodic reviews |
| **Consortium Calls** | Multi-timezone scheduling for international partners |
| **Expert Consultations** | Event types per consultation category |

### Integration

```
ConsortiaAI → Cal.com API
     │
     ├── Create event types per project phase
     ├── Embed booking widget in Partner module
     ├── Sync with milestone calendar
     └── Automated meeting prep (agenda generation via Agent)
```

### Scheduling Agent

The Scheduling Agent automates meeting coordination:

| User Input | Agent Action |
|------------|--------------|
| "Schedule kick-off with all partners" | Finds availability, creates Cal.com event |
| "When can we meet next week?" | Checks partner calendars, suggests slots |
| "Remind everyone about Friday review" | Sends calendar reminders |

## Contracts and Agreements

Full contract lifecycle management for research consortia:

### Contract Types

| Type | Description |
|------|-------------|
| **Consortium Agreement (CA)** | Main agreement between all partners |
| **Data Processing Agreement (DPA)** | GDPR-compliant data handling terms |
| **IP/Background Schedule** | Intellectual property rights allocation |
| **Non-Disclosure Agreement (NDA)** | Confidentiality terms for pre-proposal |
| **Letter of Intent (LOI)** | Partner commitment before full agreement |
| **Subcontracting Agreement** | Terms for external subcontractors |
| **Grant Agreement** | Agreement with funding body |

### Contract Workflow

| Stage | Features |
|-------|----------|
| **Templates** | Pre-approved templates per contract type |
| **Drafting** | AI-assisted clause suggestions, version tracking |
| **Negotiation** | Redline tracking, comment threads, approval workflows |
| **Signatures** | E-signature integration (DocuSign, Adobe Sign) |
| **Storage** | Secure document vault with expiry alerts |
| **Compliance** | Auto-check against funding scheme rules |

### Contract Agent

| User Input | Agent Action |
|------------|--------------|
| "Draft a DPA for partner X" | Generates DPA from template with partner details |
| "Review IP clause in section 5" | Analyzes clause, suggests improvements |
| "What are the termination terms?" | Extracts and summarizes relevant clauses |

## Financials and Payments (Stripe)

Track budgets, partner contributions, and success-based payments:

### Payment Types

| Type | Description | Trigger |
|------|-------------|---------|
| **Partner Contributions** | Cash/in-kind tracking per partner | Budget allocation |
| **Success Fees** | Milestone-based payments | Milestone completion |
| **Consultancy Fees** | Expert consultation payments | Cal.com booking |
| **License Royalties** | Post-project IP commercialization | Recurring |

### Stripe Integration

```
ConsortiaAI → Stripe API
     │
     ├── Milestone-triggered invoices
     ├── Success fee collection
     ├── Multi-partner split payments
     ├── Subscription billing (if applicable)
     └── Financial reporting dashboard
```

### Finance Agent

| User Input | Agent Action |
|------------|--------------|
| "Generate invoice for WP2 completion" | Creates Stripe invoice with milestone details |
| "What's the budget status for Q3?" | Aggregates spending, shows remaining budget |
| "Track success fee for deliverable D3.1" | Sets up milestone-triggered payment |

## Architecture

ConsortiaAI is built on the MEDrecord platform stack:

```
User → ConsortiaAI UI → Gateway → Supabase
              │              │
              ▼              ▼
        Agent Column    AI Providers
              │         (OpenAI/Anthropic)
              ▼
        Integrations
        ├── Cal.com (Scheduling)
        ├── Stripe (Payments)
        ├── DocuSign (E-Signatures)
        └── Google/Microsoft (Documents)
```

### Technology Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js 16, React, Tailwind CSS |
| Backend | Gateway API, Supabase |
| AI | AI SDK, OpenAI, Anthropic |
| Scheduling | Cal.com API |
| Payments | Stripe API |
| Documents | Google Workspace API, Microsoft Graph |
| Auth | Gateway HMAC, RBAC |

## Security and Compliance

- **Gateway Authentication** - All requests signed with HMAC
- **Role-Based Access** - Coordinator, Partner, Viewer roles per project
- **Audit Trail** - Full versioning for contracts and grant submissions
- **GDPR Compliance** - Data handling compliant with EU regulations
- **Grant Compliance** - Rules per funding scheme (Horizon, Eurostars, ZonMw, NWO)
- **PCI Compliance** - Stripe handles payment data securely
