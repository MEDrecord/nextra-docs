# Products Overview

MEDrecord offers a suite of integrated healthcare technology products designed to improve patient care, streamline operations, and ensure regulatory compliance.

## Product Portfolio

Our products work together as a unified ecosystem, sharing authentication, data, and compliance frameworks.

### Patient Engagement

- **[HealthTalk](/docs/products/healthtalk)** - Secure patient communication platform

### Operations

- **[Helpdesk](/docs/products/helpdesk)** - Internal support and ticketing system
- **[AI Agents](/docs/products/agents)** - Intelligent automation for healthcare workflows

### Research

- **[Project Management](/docs/products/project-management)** - Research project lifecycle management with AI agents

### Infrastructure

- **[Gateway](/docs/products/gateway)** - API Gateway with HMAC authentication
- **[EHR Platform](/docs/products/ehr-platform)** - Core Electronic Health Record system

## Integration Architecture

All products integrate through the Gateway, which provides:

- **Unified Authentication** - Single sign-on across all products
- **Tenant Isolation** - Multi-tenant architecture with strict data separation
- **Audit Logging** - Comprehensive audit trails for compliance
- **HMAC Security** - Cryptographic request validation

## Compliance

All products are designed to meet:

- ISO 13485 (Medical Device Quality)
- ISO 62304 (Medical Device Software Lifecycle)
- NEN 7510 (Dutch Healthcare Information Security)
- GDPR (Data Protection)

See [Compliance](/docs/compliance/overview) for detailed documentation.
