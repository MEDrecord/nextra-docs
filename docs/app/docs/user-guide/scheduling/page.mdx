# Scheduling

Manage appointment reminders and automated patient follow-ups.

## Appointment Reminders

### Automatic Reminders

Configure automated reminders that send at specified intervals:

- 1 week before appointment
- 48 hours before appointment
- 24 hours before appointment
- Day of appointment

### Manual Reminders

Send ad-hoc reminders from:

1. The appointment calendar view
2. Individual patient records
3. Scheduling queue

## Reminder Settings

### Timing Configuration

| Setting | Options |
|---------|---------|
| First Reminder | 7 days, 5 days, 3 days before |
| Second Reminder | 48 hours, 24 hours before |
| Final Reminder | Day of, 2 hours before |

### Channel Preferences

Respect patient communication preferences:

- SMS preferred
- Email preferred
- Multiple channels
- Portal only

## Confirmation Workflow

Enable appointment confirmations:

1. Patient receives reminder with confirm/cancel options
2. Patient responds via preferred channel
3. System updates appointment status
4. Staff notified of cancellations

## Follow-up Scheduling

Automate post-appointment follow-ups:

- Post-visit surveys
- Follow-up care reminders
- Prescription refill reminders
- Preventive care scheduling

## Calendar Integration

HealthTalk integrates with:

- Practice management systems
- EHR scheduling modules
- External calendar systems (via API)
