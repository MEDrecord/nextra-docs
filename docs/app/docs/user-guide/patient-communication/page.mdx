# Patient Communication

Learn how to communicate effectively and securely with patients using HealthTalk.

## Sending Messages

### Individual Messages

1. Navigate to **Messages > New Message**
2. Search for the patient by name, ID, or contact information
3. Select the communication channel (Email, SMS, Portal)
4. Choose a template or compose a custom message
5. Preview and send

### Bulk Messages

For sending to multiple patients:

1. Navigate to **Messages > Bulk Send**
2. Upload patient list or select from a saved group
3. Choose template with merge fields
4. Preview sample messages
5. Schedule or send immediately

## Message History

All messages are logged and accessible via:

- Patient record view
- Message history search
- Audit trail reports

## Response Handling

When patients respond:

1. Notifications appear in your inbox
2. Responses are linked to the original message thread
3. Assign to appropriate staff member if needed
4. Document any clinical follow-up required

## Compliance Notes

- All messages are encrypted in transit and at rest
- Full audit trail is maintained for regulatory compliance
- Message content must comply with organizational policies
- PHI should only be shared through secure channels
