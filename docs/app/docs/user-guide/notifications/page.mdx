# Notifications

Configure and manage system notifications for staff and administrators.

## Notification Types

### Message Notifications

- New patient responses
- Message delivery failures
- Urgent messages flagged for review

### System Notifications

- User account activities
- System maintenance alerts
- Compliance reminders
- Template approval requests

### Administrative Notifications

- Usage threshold alerts
- Security events
- Integration status changes

## Notification Preferences

### Personal Settings

Configure your notification preferences:

1. Navigate to **Settings > Notifications**
2. Select notification categories
3. Choose delivery method (in-app, email, SMS)
4. Set quiet hours if needed

### Notification Channels

| Channel | Use Case |
|---------|----------|
| In-app | Real-time alerts while working |
| Email | Digest and important notifications |
| SMS | Critical alerts and escalations |

## Notification Center

Access all notifications from the bell icon in the header:

- View unread notifications
- Mark as read/unread
- Filter by category
- Search notification history

## Team Notifications

Administrators can configure team-wide notifications:

- Assign notification responsibilities
- Set up escalation rules
- Configure on-call rotations
- Define notification groups
