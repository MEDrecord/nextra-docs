## Link Replacement Patterns

When updating links, use these patterns:

### Azure DevOps to Helpdesk
```mdx
// Before
[Azure wiki page](https://dev.azure.com/medrecord/MEDrecord%20ISMS/_wiki/wikis/MEDrecord-ISMS.wiki/1174/Information-retention-policy)

// After
[Information Retention Policy](/helpdesk/isms/information-retention-policy)
```

### Atlassian to Internal ISMS
```mdx
// Before
[A.16.1.2 Reporting information security events (NEN 7510)](https://medrecord.atlassian.net/wiki/spaces/ISMS/pages/8486968)

// After
[A.16.1.2 Reporting information security events](/isms/controls/a-16-1-2)
```

### Empty Placeholder Fix
```mdx
// Before
- Implement ,
- Implement  and 

// After
- Implement [Password Policy](/helpdesk/isms/password-policy)
- Implement [Access Control Policy](/helpdesk/isms/access-control-policy) and [Information Classification Policy](/helpdesk/isms/information-classification-policy)
```
