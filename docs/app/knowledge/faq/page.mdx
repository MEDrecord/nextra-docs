# Frequently Asked Questions

## General

### What is HealthTalk?
HealthTalk is a secure patient communication platform that enables healthcare providers to communicate with patients through multiple channels while maintaining compliance with healthcare regulations.

### Which regulations does the platform comply with?
The platform is compliant with ISO 13485, ISO 62304, NEN 7510, and GDPR. See the Compliance section for detailed documentation.

### How is patient data protected?
All patient data is encrypted at rest and in transit. Access is controlled through role-based permissions, and all actions are logged in an immutable audit trail.

## Account and Access

### How do I reset my password?
Contact your organization administrator or use the password reset link on the login page.

### What are the supported browsers?
We support the latest versions of Chrome, Firefox, Safari, and Edge.

### Can I access the platform on mobile devices?
Yes, the platform is responsive and works on mobile browsers. Native apps are available for iOS and Android.

## Messaging

### What message channels are supported?
SMS, email, in-app notifications, and patient portal messaging.

### Are messages encrypted?
Yes, all messages are end-to-end encrypted.

### Can I schedule messages?
Yes, messages can be scheduled for future delivery through the Scheduling feature.

## Integration

### Which EHR systems are supported?
We integrate with major EHR systems via HL7 FHIR and openEHR standards. Contact support for specific EHR compatibility.

### How do I set up API access?
See the API Reference section in the Developer documentation for setup instructions and authentication details.
